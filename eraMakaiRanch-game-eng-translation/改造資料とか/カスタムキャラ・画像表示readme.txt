v0.3.02よりcsvファイルからのカスタムキャラクター読み込み、
resourcesフォルダ内の画像を使っての顔グラフィック表示ができるようになりました。

・カスタムキャラクターを作成する
①母乳を搾りたいキャラクターを決めます。

②「CSV」フォルダを開き、「Chara2_サンプル.csv」ファイルをフォルダ内にコピー、
コピーしたファイルの名前を「Chara3_（作成するキャラクター名）.csv」に変更します。
（わかりやすくするため）

③名前を変更したcsvファイルを
【重要】「テキストエディタにて」開いてください。
よくわからない場合、今このtxtファイルを開いている部分にcsvファイルを
ドラッグ＆ドロップすればテキストエディタで開くことができると思います。

ファイルを開くことができたら、同じ方法で「Talent.csv」や「Str.csv」も見ることで
キャラクター作成の参考になると思います。
（Talent＝素質　はガワだけ作った状態がほとんどなので、フレーバーだと思うほうがいいでしょう。）

④コピーしたcsvファイルに書かれたことを読みながらデータを入力していってください。
（説明を入れながらなので非常に長いですがご容赦を）

⑤入力が終わったら上書き保存し、エラーを出さずに起動するのを確認したら
思う存分搾乳してください。



・顔グラフィックを表示する

①母乳を搾りたいキャラクターの顔グラフィックを用意します。
個人で楽しむ分にはどこから持ってきてもOKです。
本体に同梱するのはちょっとまずいかなと思うため表示確認用のサンプルしか入っていません。

現在「ステータス画面用の画像」、「調教のコマンド待機画面用の画像」、
「調教コマンド実行後の画像」の最大3種類を表示することができます。
服の状態とか膨乳には対応していない（対応しないのでなんでも持ってこれるとも言う）ため、
顔だけ表示するのが破綻しなくていいと思います。

②「resources」フォルダを開き、「Resources.csv」ファイルを
【重要】「テキストエディタにて」開いてください。

③開いたcsvファイルに書かれたことを読みながらデータを入力していってください。

④入力が終わったら上書き保存し、エラーを出さずに起動するのを確認したら
思う存分搾乳してください。

①「Resources.csv」に従いつける識別名に対応した画像を準備する

②画像を表示したいキャラの「識別名」を変更し、画像が表示されるか確認する
