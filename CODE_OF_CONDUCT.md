# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while giving clear feedback.
- Focus discussion on the code and project goals.

## Unacceptable Behavior

- Harassment, abuse, or hateful conduct.
- Sexualized language toward contributors.
- Doxxing, threats, or intimidation.
- Repeated disruptive behavior after warnings.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this Code of Conduct.

## Reporting

Report conduct issues privately to the maintainers through repository security/contact channels.
